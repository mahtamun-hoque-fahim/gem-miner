Why Gem Miner

Everything you'd want from a chat export

Built for developers and researchers who need to keep records — not just screenshots.

Clean PDF output
Uses the browser's native print engine — no pixel-shifted screenshots, no bloated canvas renders. Crisp, searchable, indexable text.
Markdown preserved
Code blocks, tables, bold, italic, headers, blockquotes — Gemini's rendered HTML is sanitized and re-styled in the export, not flattened.
Light + dark themes
Choose a white page for print or dark background for screen. Both themes ship with proper code block contrast, brand colors, and A4 margins.
Thinking blocks skipped
Gemini 2.5's reasoning traces are automatically excluded. Only the final response lands in your PDF — no noise, no internal monologue.
Cross-browser
Manifest V3. Works in Chrome, Edge, Brave, Arc, and Firefox 109+. One codebase, no platform-specific hacks, no Node.js required.
Zero dependencies
No npm, no bundler, no jsPDF, no html2canvas. Just a manifest and four plain JS files you can read in an afternoon.

What is a decorator in Python and when should I use one? Can you show me an example of a timing decorator?

Gemini

A decorator is a function that wraps another function to extend or modify its behavior — without permanently changing the original function's source code.

Here's a practical timing decorator:

import time

def timer(func):
    def wrapper(*args, **kwargs):
        start = time.perf_counter()
        result = func(*args, **kwargs)
        elapsed = time.perf_counter() - start
        print(f"{func.__name__} ran in {elapsed:.4f}s")
        return result
    return wrapper

@timer
def slow_operation():
    time.sleep(1)
Get started

Install Gem Miner

The extension loads unpacked in under a minute. No account, no sign-in, no data leaves your browser.

Download gem-miner.zip
1
Download and unzip the file above into a permanent folder — don't delete it after installing.
2
Open chrome://extensions (or edge://extensions / brave://extensions).
3
Enable Developer mode using the toggle in the top-right corner.
4
Click Load unpacked and select the gem-miner-main folder you unzipped.
5
Pin the Gem Miner icon to your toolbar via the puzzle-piece extensions menu. Done.
The extension only has permission to run on gemini.google.com. No other sites, no background data collection.
Download gem-miner.zip
1
Download and unzip the file above.
2
Open about:debugging in Firefox.
3
Click This Firefox in the left panel, then Load Temporary Add-on…
4
Navigate into the gem-miner-main folder and select the manifest.json file.
Temporary add-ons in Firefox are removed when you restart the browser. For a persistent install, the extension must be signed via Mozilla Add-on Hub. A signed release is planned for v1.1.
